## Installation

*On Your Jetbrains IDE*:

- <kbd>Open Settings/Preferences</kbd>
- <kbd>Plugins</kbd>
- <kbd>Marketplace</kbd>
- <kbd>Search for "TokyoNight Theme"</kbd> >
- <kbd>Install Plugin</kbd>
